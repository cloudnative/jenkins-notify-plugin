Build result for which a notification will be send down to. So notifications will be send for all build results which are equal or better than the configured one.

  • SUCCESS - Notification gets send only for SUCCESS results
  • UNSTABLE - Notification gets send for UNSTABLE and SUCCESS results
  • FAILURE - Notification gets send only for FAILURE, UNSTABLE and SUCCESS results
  • NOT_BUILT - Notification gets send only for NOT_BUILT, FAILURE, UNSTABLE and SUCCESS results
  • ABORTED - Notification gets send only for ABORTED, NOT_BUILT, FAILURE, UNSTABLE and SUCCESS results
